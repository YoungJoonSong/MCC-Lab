# Mobile Communications and Coding Lab

## 개요
Mobile Communications and Coding Lab(MCC Lab)은 금오공과대학교 전자공학부에 소속된 연구실입니다. 차세대 이동통신 시스템과 채널 부호화 이론, 그리고 인공지능 기술의 연구와 개발을 중점적으로 수행하고 있습니다.

## 연구 분야
- **5G/Beyond 5G 이동통신 시스템**: 차세대 이동통신 기술 연구
- **채널 부호화 이론**: 통신 시스템의 신뢰성 향상을 위한 부호화 기술
- **인공지능 및 머신러닝**: AI/ML 알고리즘 개발 및 응용
- **자율주행 시스템**: 머신러닝을 활용한 자율주행 기술
- **공공안전 시스템**: AI 기반 공공안전 체계 구현
